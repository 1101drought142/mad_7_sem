# Лабораторная работа: Классификация ирисов

## Описание проекта

Проект реализует полное решение задачи классификации на датасете Iris Plants Database с использованием двух методов:
- **Нейронная сеть (MLPClassifier)**
- **Метод опорных векторов (SVM)**

## Что делает скрипт

Скрипт выполняет все этапы задания:

1. ✅ Загрузка и описание данных
2. ✅ Дескриптивный анализ и проверка нормальности распределения
3. ✅ Оценка распределения объектов по классам
4. ✅ Отбор наиболее информативных признаков
5. ✅ Стандартизация переменных
6. ✅ Построение категоризованных диаграмм рассеивания
7. ✅ Разделение на обучающее и тестовое подмножества
8. ✅ Построение моделей классификации (MLP и SVM)
9. ✅ Оценка качества моделей (confusion matrix, accuracy, precision, recall, F1, AUC)
10. ✅ Оценка значимости признаков
11. ✅ Изменение параметров методов и анализ результатов
12. ✅ Удаление малозначимых признаков и переоценка

## Описание датасета

**Iris Plants Database**
- 150 записей (по 50 в каждом из 3 классов)
- 4 признака: sepal length, sepal width, petal length, petal width
- 3 класса: Iris-setosa, Iris-versicolor, Iris-virginica
- Один класс (setosa) линейно отделим от остальных двух

## Методы классификации

### 1. Нейронная сеть (MLPClassifier)
- **Архитектура**: 4 входных → 2 скрытых слоя (100, 50 нейронов) → 3 выходных
- **Активация**: ReLU
- **Оптимизатор**: Adam
- **Регуляризация**: L2 (alpha=0.001)

### 2. Метод опорных векторов (SVM)
- **Ядро**: RBF (радиальная базисная функция)
- **C**: 1.0
- **Gamma**: 'scale' (автоматический выбор)

## Метрики качества

Для каждой модели рассчитываются:
- Accuracy (точность)
- Precision (точность) - macro и micro усреднение
- Recall (полнота) - macro и micro усреднение
- F1-score - macro и micro усреднение
- AUC-ROC - macro и micro усреднение
