Цель работы: изучение методов решения задачи классификации.
Платформа: по выбору студента (RStudio, python, SPSS, Loginom и др.).
Задание
1. Выбрать массив данных (ссылки № 1-2 из списка литературы),
описать параметры. Примечание: тип задачи – «классификация».
2. Провести дескриптивный анализ; определить, согласуются ли
выборки с нормальным распределением. Оценить распределение
объектов по классам. Аргументировать выбор методов решения
задачи классификации по результатам описания датасета.
3. Провести отбор наиболее информативных признаков.
4. Стандартизировать переменные (при необходимости).
5. Построить категоризованные диаграммы рассеивания, оценить
расположение классов.
6. Разделить данные на обучающее и тестовое подмножества.
7. Решить задачу классификации двумя методами (нейронная сеть,
дерево решений, метод опорных векторов, дискриминантная
функция и т.д.). Аргументировать выбор значений параметров
метода.
8. Оценить качество построенных моделей. Проанализировать confusion
matrix, метрики на тестовом и обучающем множествах (accuracy,
precision, recall, AUC, F-мера; макро- и микроусреднение). Провести
сравнительный анализ решений.
9. Оценить значимость признаков.
10. Изменить параметры одного из методов, оценить полученные
результаты.
11. В случае выявления малозначимых признаков – удалить их из
массива; оценить изменение точности классификатора при решении
задачи с исключенными из датасета признаками (для одного из
методов).
12. Оформить отчет.
Содержание отчета
1. Титульный лист.
2. Цель работы.
3. Описание исходных данных.
4. Результаты дескриптивного анализа.
5. Описание параметров метода (в т.ч. структуры нейросети при выборе
данного метода решения).
6. Результаты решения задачи классификации (confusion matrix,
метрики, значимость признаков). При выборе в качестве метода
решения дерева решения – иллюстрация дерева с указанием
индексов Gini (энтропии) в узлах.
7. Сравнительный анализ решений и интерпретация результатов.
8. Результаты анализа влияния параметров одного из методов на
точность классификатора.
9. Результаты сравнительного анализа решений в случае исключения
малозначимых признаков.

Вопросы для подготовки к защите лабораторной работы. Примеры
практических заданий.
1. Постановка задачи классификации. Ограничения, накладываемые
на исходные данные. Требования к исходным данным при
использовании того или иного метода. Бинарная классификация.
Многоклассовая классификация.
2. Методы решения задачи классификации. Метод k ближайших
соседей (kNN). Метод опорных векторов. Деревья решений.
Комитетные методы, стратегии построения обучающего множества.
Леса (джунгли). Bagging. Boosting.
3. Нейронная сеть в задаче распознавания: типы, структура, виды
активационных функция, методы обучения. Метод обратного
распространения ошибки.
4. Дискриминантный анализ. Дискриминантная функция.
5. Процедура применения алгоритма (метод опорных векторов,
дискриминантная функция, метод k-ближайших соседей, дерево
решений).
6. Метрики качества классификации. Доля правильно распознанных
ответов, точность, полнота и др. PR-, ROC-кривая. Оценка качества
многоклассовой классификации.
7. Заполнить confusion matrix, рассчитать метрики (accuracy, precision,
recall, F-меру и др.).
8. По представленным результатам решения задачи классификации
оценить точность классификатора, значимость признаков. Оценить
метрики (в т.ч. AUC).
9. Алгоритм построения дерева решений. Для представленного
массива построить фрагмент дерева решений. Рассчитать индекс
неоднородности для узлов дерева. Оценить по индексу
неоднородности качество дерева.
10. Оценка значимости признаков
